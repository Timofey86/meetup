# Meetup Clone (Frontend Project)

This project is a frontend implementation of a Meetup-like interface based on a Figma design.

It includes a main page and an events page with dynamic filtering.

## Tech Stack

- HTML
- CSS (Flexbox, Grid)
- JavaScript (Vanilla)

## Features

- Responsive layout (desktop, tablet, mobile)
- Event list rendered from mock data
- Filtering by:
    - Date
    - Category
    - Distance
    - Type (Online / Offline)
- Google Maps embedded via iframe

## Notes

- Filtering is implemented using native JavaScript array methods.
- Filters are applied cumulatively
- Map integration is done without external API (iframe)

## Purpose

This project was created to practice frontend fundamentals and UI implementation from design.
